# Welcome

This UI Library is the Arch Capital design system. It includes styles, components, and patterns for various UI elements within the Arch Capital Group website. This library is intended to be used as a reference and guide for the design and development teams when updating or creating new pages on the Arch Capital Group website.

## What's Included

### Primitives

The foundational styles and components that are used heavily throughout the UI library. This includes colors, typography, spacing, and more.

### Controls & Inputs

These components allow users to input data and perform actions within the site. This includes things like buttons, text inputs, selects, and more.

### Blocks

The main function of these components is to display information. This includes things like Call to Action, Hero, Summary Card, and more.

### Navigation

Components that allow users to navigate around the site. This includes Global Navigation, Main Navigation, and more.
